# 🍽️ Recipe App

A simple and interactive **Recipe App** built using **HTML, CSS, and JavaScript**. This app allows users to search and explore recipes, view ingredients, and follow step-by-step cooking instructions.

---

## 🚀 Features

- 🔍 Search for recipes  
- 📋 View ingredients list  
- 🧑‍🍳 Step-by-step instructions  
- 🎨 Responsive and clean UI  
- ⚡ Lightweight and fast  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
